Each button below triggers a different type of inline CSP violation. Watch the log panel and your server's stdout to confirm reports are arriving.

script-src · inline

Injected <script> tag

Creates a script element with inline code and appends it to the DOM. Violates script-src 'self'.

style-src · inline

Injected <style> tag

Appends a <style> element to the <head>. Violates style-src 'self'.

script-src · eval

eval() call

Calls eval() with a string expression. Violates script-src without 'unsafe-eval'.

script-src · handler

setAttribute onclick

Sets an onclick attribute via JavaScript. Treated as inline script by strict CSP.
